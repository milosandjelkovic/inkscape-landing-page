Inkscape

Inkscape is a free vector graphics editor similar to Adobe Illustrator, Corel Draw, or Affinity Designer. What sets Inkscape apart is its use of Scalable Vector Graphics (SVG) as the native format

For designers of all kind

If you are new to the process of creating vector graphics it may feel different, but you will quickly be pleased by the flexibility, and power Inkscape offers. The Inkscape application is used across a wide variety of industries (marketing/branding, engineering/CAD, web graphics, cartooning) and individual uses.

Professional Publishing

All Inkscape projects may be exported in formats friendly to web browsers or commercial printer rooms. It is cross-platform, which means it is easy to run on Windows, Mac OS X, and Linux distributions. Visit the Download page to install or share this application now

Features of Inkscape

Flexible drawing tools

  • Drawing: pencil tool, pen tool, calligraphy tool
  • Shape tools: rectangles, stars/polygons, spiral
  • Embedded bitmaps (with a command to create and embed bitmaps of selected objects
  • Clones ("live" linked copies of objects), including a tool to create patterns and arrangements of clone

Broad file format compatibility

  • Perfectly compliant SVG format file generation and editing
  • Live watching and editing the document tree in the XML edito
  • PNG, OpenDocument Drawing, DXF, sk1, PDF, EPS and PostScript export formats and mor
  • Clones ("live" linked copies of objects), including a tool to create patterns and arrangements of clone

Powerful text tool

  • Multi-line text
  • Kerning, letterspacing, linespacing adjustment
  • Text on path (both text and path remain editable
  • Text in shape (fill shape following stroke)
